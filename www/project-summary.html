Paper

A concise overview plus a link to the full manuscript.

A Dynamic Risk Prediction Model for Rainfall-Triggered Landslides in Buncombe County, North Carolina
Read the paper (PDF)

  • Goal: Localized, near-real-time risk model fusing daily rainfall windows with terrain and soils.
  • Data: 302 landslides (1981–2024) + 302 controls; PRISM daily rainfall windows (R1d, R3d, R7d, R30d, Max 3-day/30-day); USGS DEM (elevation & slope); SSURGO soil depth.
  • Models: Logistic Regression, Random Forest, XGBoost; stratified CV with spatial blocking; SHAP for interpretation.
  • Findings: Slope dominates. Short-duration/intense rainfall boosts risk; prolonged 30-day accumulation can show a negative association with immediate triggering.
  • Best overall: Random Forest with terrain + rainfall (~0.81 accuracy, ~0.89 AUC).

How the model works

  • Daily PRISM precipitation is summarized into 1–30-day windows and rolling maxima.
  • Static terrain (elevation, slope) and soil depth are sampled on a county grid.
  • Random Forest outputs per-cell probabilities (p_rf), updated as new rainfall arrives.
  • Outputs are written to predictions/historicData/YYYY/MM/ (Parquet, GeoJSON, meta).

What this website provides

  • Latest map: /predictions/latest.geojson for immediate viewing.
  • Historical runs: Date-addressable GeoJSON & Parquet under /predictions/historicData/....
  • Manifest: /predictions/index.json lists all available dates for the “Runs” picker.
  • Reproducibility: Each run has meta_YYYY-MM-DD.json with counts, date, features, and model info.

How to interpret it probabilities

  • What a probability means: Values are learned from a balanced dataset (302 landslides vs 302 non-events, 1981–2024). A value is the model’s estimated likelihood that current terrain + rainfall conditions resemble historical triggering conditions.
  • Susceptibility vs forecast: On any given day, if you refresh rainfall windows (R1d–R30d, Max3d/30d) with current PRISM data, the map functions as a nowcast of instability conditioned on today’s hydrometeorology. It is not a multi-day forecast and should be interpreted as short-term (≈24 h) failure likelihood given present conditions.
  • Reading the scale:
    • 0.0–0.3 — low: conditions unlike past failures (generally stable)
    • 0.4–0.6 — uncertain/watch: near historical thresholds; monitor during ongoing rain
    • 0.7–1.0 — high: conditions closely match past triggers; prioritize attention
  • Thresholds: The default visual cutoff of 0.5 is for display only. For operations, choose a threshold via ROC/PR curves (e.g., Youden’s J) or by cost (favor recall for safety-critical alerts).
  • Calibration: To treat values as actionable probabilities, evaluate reliability (e.g., Brier score, reliability curves) on recent years and apply Platt or isotonic calibration if needed.
  • Caveats: Map indicates likelihood, not consequence (size/runout). Resolution limits (PRISM grid), reporting bias in historical inventories, and terrain changes (construction, vegetation) can affect accuracy.
  • Example: A pixel with p=0.4 means moderate susceptibility today—some failure-like signals (slope/rainfall) but below the typical triggering threshold. Under additional rainfall, it may cross into the watch/high zone.
