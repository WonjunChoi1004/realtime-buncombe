Paper

A concise overview plus a link to the full manuscript.

A Dynamic Risk Prediction Model for Rainfall-Triggered Landslides in Buncombe County, North Carolina
Read the paper (PDF)

  • Goal: Localized, near-real-time risk model fusing daily rainfall windows with terrain and soils.
  • Data: 302 landslides (1981–2024) + 302 controls; PRISM daily rainfall windows (R1d, R3d, R7d, R30d, Max 3-day/30-day); USGS DEM (elevation & slope); SSURGO soil depth.
  • Models: Logistic Regression, Random Forest, XGBoost; stratified CV with spatial blocking; SHAP for interpretation.
  • Findings: Slope dominates. Short-duration/intense rainfall boosts risk; prolonged 30-day accumulation can show a negative association with immediate triggering.
  • Best overall: Random Forest with terrain + rainfall (~0.81 accuracy, ~0.89 AUC).

How the model works

  • Daily PRISM precipitation is summarized into 1–30-day windows and rolling maxima.
  • Static terrain (elevation, slope) and soil depth are sampled on a county grid.
  • Random Forest outputs per-cell probabilities (p_rf), updated as new rainfall arrives.
  • Outputs are written to predictions/historicData/YYYY/MM/ (Parquet, GeoJSON, meta).

What this website provides

  • Latest map: /predictions/latest.geojson for immediate viewing.
  • Historical runs: Date-addressable GeoJSON & Parquet under /predictions/historicData/....
  • Manifest: /predictions/index.json lists all available dates for the “Runs” picker.
  • Reproducibility: Each run has meta_YYYY-MM-DD.json with counts, date, features, and model info.
